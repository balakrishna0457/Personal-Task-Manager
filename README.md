# Task Manager Web Application

A web-based productivity tool designed to streamline task organization and enhance efficiency. This application acts as a personal productivity assistant by allowing users to manage tasks, track progress, and receive automated email reminders.

## üöÄ Features
- **User Authentication:** Secure login and registration system to protect user data.
- **Task Management:** Full CRUD functionality (Add, View, Update, Delete) for personal tasks.
- **Automated Notifications:** Sends email alerts via external APIs when a task's scheduled time arrives.
- **Real-time Tracking:** Dynamic countdown timers for pending tasks using JavaScript.
- **Task History:** Dedicated module to view a complete history of task creation, completion dates, and status.
- **Responsive UI:** Intuitive and user-friendly interface built for a seamless experience.
